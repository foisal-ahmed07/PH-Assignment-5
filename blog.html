Important FAQ

1. What is the Document Object Model (DOM)?

1. The DOM is a programming interface representing a webpage as a structured tree, allowing dynamic manipulation of content, styles, and structure.

2. How do you select an element from the DOM?

1. Use methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName() to select elements from the DOM.

3. What is event delegation in the context of the DOM, and why is it useful?

1. Event delegation allows event handling on a parent element, capturing events from child elements, improving performance and simplifying code for dynamically added elements.

4. How do you manipulate an element's attributes and styles using the DOM?

1. You can manipulate attributes using element.setAttribute()/element.getAttribute() and styles using element.style.property.
